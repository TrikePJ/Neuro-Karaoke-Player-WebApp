### How to Install

### Via pre build installer
1. **Download the installer** from the [Releases](https://github.com/TrikePJ/Neuro-Karaoke-Player-Test/releases) page
2. **Run the installer**
3. The Player opens right after installation

### Build it yourself

1. **Install Node.js**
   - Download https://nodejs.org/

2. **Clone the repo**
   ```bash
   git clone https://github.com/TrikePJ/Neuro-Karaoke-Player-Test.git
   ```

4. **Install Dependencies**
   ```bash
   npm install
   ```

5. **Build the Windows installer:**
   ```bash
   npm run build:win
   ```
6. You can find the installer in the `Installer` folder:
   - `Neuro Karaoke Player Setup.exe`

   ## Features

-  Neuro Karaoke website by soul in a native desktop window
-  Discord login works with persistent sessions
-  Closing the App hides to Sys Tray
